☁️ Google 雲端硬碟同步 (像 Google Sheet 一樣自動雙向存取)

將所有的租屋資料（rental_helper_db.json）儲存在您自有的 Google Drive 雲端硬碟中，安全隱私不外洩給第三方。 任何一台裝置（包含家人的手機）只要登入同一個 Google 帳號，或是由擁有者將雲端硬碟中的該檔案分享給家人的 Google 帳號，就能實現多人共同編輯、自動即時同步！

📋 5 分鐘快速設定教學（100% 個人資料隱私自主）：
  1. 前往 Google Cloud 主控台 並登入您的 Google 帳號（例如 youngyang@17.media）。
  2. 建立新專案（例如命名為 Rental Helper）。
  3. 在左側選單選擇「API 和服務」>「OAuth 同意畫面 (OAuth consent screen)」：
    • User Type 選擇「外部 (External)」（若使用 Workspace 帳號亦可選內部）。
    • 填寫應用程式名稱（例如 張爸爸房租助手）與您的聯絡信箱。
    • 在「範圍 (Scopes)」步驟中，點選「新增或移除範圍」，手動在底部貼入並新增：
      https://www.googleapis.com/auth/drive.file
      https://www.googleapis.com/auth/userinfo.email
      （用於讀寫本程式產生的雲端檔案與確認登入身分，不影響您硬碟中其他私密檔案）。
    • 在「測試使用者 (Test users)」步驟中，務必新增您與家人的 Google 信箱，這樣大家才能順利登入！
  4. 左側選單選擇「憑證 (Credentials)」，點選上方「+ 建立憑證」>「OAuth 2.0 用戶端 ID」：
    • 應用程式類型選擇「網頁應用程式 (Web application)」。
    • 在 已授權的 JavaScript 來源 中新增兩個網址：
      🔹 本機測試：http://localhost:5000 或 http://127.0.0.1:5000 (若有在 local 測試)
      🔹 線上網址：https://shyshyhao.github.io
  5. 建立後，複製產生的 用戶端 ID (Client ID) 並貼到上方的輸入框中，點選下方「儲存 Client ID」。
  6. 👨‍👩‍👧‍👦 家庭協同編輯設定：
    當您登入並成功同步後，您的雲端硬碟根目錄會自動產生一個 rental_helper_db.json 檔案。 請您登入 Google 雲端硬碟 (Google Drive) 網頁版，將該檔案共用（Share）給您家人的 Google 帳號，並給予「編輯者」權限。
    接著家人在他們的手機點開網頁、填入同一個 Client ID 並登入其 Google 帳號，就會自動讀寫這份共用的 JSON 檔案，像 Google Sheet 一樣實現多人即時同步編輯！

🔗 家人多裝置同步與分享 (舊版 LINE 連結同步)

iPhone 開啟本機網頁時，瀏覽器資料常因快取被清除而遺失。本功能可將目前所有的房客、設定、歷史帳單打包成一個同步連結。貼到 LINE 分享給家人，點擊即可一鍵覆蓋同步！

🔒 網頁安全存取密碼鎖

啟用後，當您（或拿到分享連結的其他人）開啟網頁時，必須輸入正確的密碼才能看見並編輯租屋內容。適合部署到 GitHub Pages 公開網頁時使用，防止無權限的陌生人窺看！

💾 資料備份與恢復

所有的資料皆安全的保存在您目前的手機/瀏覽器中。建議定期備份以防手機遺失或清除快取。
